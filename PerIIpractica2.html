Introducción del método del contexto, EntityFramework y LinQ.

Para la creación de los métodos de Marca el primer punto a tocar para que el proyecto funcione de forma normal es importar las librerías de EntityFramework y LinQ.

Luego de hacer la importación de las librerías se debe de verificar que tenga lo que se hizo en la práctica pasa, lo cual es tener la base de datos conectada con el proyecto y que este reconozca lo que se consultara, para ello se crea el contexto que nos importa la información que la aplicación necesita para solicitar consultas a la base de datos:

Método de consulta GET en general

• Controlador de Marca

Luego de verificar esto ahora se procede a realzar la consulta, primero le especificamos que es una consulta Get con (HttpGet), luego se coloca la ruta con la que se consultara y hacemos una consulta con LinQ donde le pedimos que nos traiga todos los datos de marcas y un IF donde compara si esta lista es mayor a 0, si es mayor nos retornara los datos y si no nos retornara un informe de error dándonos información de este.

Para los demás controladores se realiza exactamente lo mismo con la variación de la ruta, el contexto y el modelo al que se realiza el llamado.

Método de Búsqueda por ID

Para el método de búsqueda por ID se realizó como primer punto la especificación del HttpGet para especificar qué tipo de método es y luego la ruta que se utilizara para la consulta agregando un /{id} que es lo que nos permitirá hacer la consulta por el ID, dado que el método nos solicitara un ID que es el que se coloca en la URL.

Luego se realiza la consulta enviando el contexto la clase del modelo y se realiza el filtro donde compara si el id enviado es igual a uno de la base de datos y con FirstOrDefault(); devuelve el primer elemento de la secuencia.

Luego con un IF se compara si este valor existe mediante el Null o existe, de existir nos envía la colección del JSON y si no existe nos envía el informe de errores. Y de igual forma es similar para los otros controladores.
